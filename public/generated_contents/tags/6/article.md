# 3次元点群 (3D Point Cloud)

3次元点群（3D Point Cloud）は、LiDARやステレオカメラ等で取得した空間中の点の集合データです。当研究室では、点群データを用いた人物検出・行動認識・電子透かし技術を研究しています。

3D Point Cloud is a collection of points in space acquired by LiDAR, stereo cameras, etc. Our laboratory researches person detection, action recognition, and digital watermarking technologies using point cloud data.
