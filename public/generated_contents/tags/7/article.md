# 機械学習 (Machine Learning)

機械学習は、データからパターンを学習し予測・分類を行う技術です。当研究室では、交通流予測や人物行動認識、通信ネットワーク最適化への機械学習応用を研究しています。

Machine Learning is a technology that learns patterns from data to perform prediction and classification. Our laboratory researches applications of machine learning in traffic flow prediction, human action recognition, and communication network optimization.
