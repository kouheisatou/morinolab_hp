# センシング (Sensing)

センシング技術は、各種センサを用いて物理世界の情報をデジタルデータとして取得する技術群です。Wi-Fi CSI、LiDAR、IMU等の多様なセンサを活用した応用研究を行っています。

Sensing technologies are a set of technologies for acquiring physical world information as digital data using various sensors. We conduct application research using diverse sensors including Wi-Fi CSI, LiDAR, and IMU.
